URACS
=====

统一角色访问控制系统(Unified Role Access Control System)，是基于Spring Security 3实现的权限控制系统

<br/>
程序框架版本说明：Spring MVC 3.2.0 + Spring Security 3.1.3 + Hibernate 3.6.10


<br/>
开发者：李锡远
<br/>
博客：http://taven.cnblogs.com
